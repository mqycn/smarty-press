# Smart-Press
- 快速高效的 Markdown 网站制作工具
- 基于 Vue3.0 SSR 技术

## 交流群

![qrcode](assets/qrcode-2216750.JPG)

## 起步

Manually
```bash
# install 
npm install smarty-press -g

# start to writing 
spress start

# build to html
spress build

```


## 开发

```js
# clone first
npm link
cd demo
spress start
```

```bash
yarn add vue@3.0.0

```
